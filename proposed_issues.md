# Предлагаемые задачи (Issues) для репозитория skills-integrate-mcp-with-copilot

На основе анализа репозитория ClubPortalEvent, вот список новых функций, которые можно добавить к нашему проекту (FastAPI-приложению для управления внеучебными мероприятиями). Каждая функция описана как потенциальная задача (issue) с заголовком, описанием и метками.

## 1. Добавить мобильную платформу (Android-приложение)
**Заголовок:** Add mobile Android app support  
**Описание:** Создать мобильное приложение на Android для доступа к мероприятиям, аналогично ClubPortalEvent. Это позволит студентам регистрироваться и управлять мероприятиями на ходу.  
**Метки:** enhancement, mobile, android  
**Приоритет:** High  

## 2. Реализовать аутентификацию и роли пользователей
**Заголовок:** Implement user authentication and roles (Student, Admin, Super Admin)  
**Описание:** Добавить систему аутентификации (например, через Firebase Auth или JWT) с ролями: студенты могут регистрироваться, администраторы управляют мероприятиями, супер-админы управляют клубами.  
**Метки:** authentication, roles, security  
**Приоритет:** High  

## 3. Перейти на постоянную базу данных
**Заголовок:** Migrate from in-memory to persistent database (e.g., Firestore or PostgreSQL)  
**Описание:** Заменить in-memory словарь на постоянную базу данных для сохранения данных между перезапусками сервера.  
**Метки:** database, persistence  
**Приоритет:** High  

## 4. Добавить поддержку изображений и медиа (постеры мероприятий)
**Заголовок:** Add support for event posters and images  
**Описание:** Позволить загружать и отображать изображения постеров для мероприятий, чтобы сделать интерфейс более привлекательным.  
**Метки:** ui, media, images  
**Приоритет:** Medium  

## 5. Внедрить QR-коды для регистрации и билетов
**Заголовок:** Implement QR code generation for event registration tickets  
**Описание:** После регистрации генерировать QR-код для билета, который можно сканировать для подтверждения.  
**Метки:** qr-code, registration, mobile  
**Приоритет:** Medium  

## 6. Добавить отслеживание посещаемости с QR-сканированием
**Заголовок:** Add attendance tracking with QR code scanning  
**Описание:** Реализовать сканирование QR-кодов для отметки присутствия, с временными ограничениями и экспортом в CSV.  
**Метки:** attendance, qr-code, export  
**Приоритет:** Medium  

## 7. Внедрить управление бюджетом с OCR для чеков
**Заголовок:** Implement budget management with OCR for receipts  
**Описание:** Добавить отслеживание доходов/расходов, сканирование чеков с распознаванием суммы (используя ML Kit или аналог) и визуализацию с диаграммами.  
**Метки:** budget, ocr, charts  
**Приоритет:** Medium  

## 8. Добавить управление задачами (чек-листы для мероприятий)
**Заголовок:** Add task management (checklists) for events  
**Описание:** Позволить создавать и отмечать задачи для каждого мероприятия (например, подготовка, логистика).  
**Метки:** tasks, organization  
**Приоритет:** Low  

## 9. Реализовать обратную связь с анализом настроения
**Заголовок:** Implement feedback system with sentiment analysis  
**Описание:** Добавить форму для отзывов после мероприятия с автоматическим анализом настроения (положительный/отрицательный/нейтральный).  
**Метки:** feedback, ai, sentiment  
**Приоритет:** Low  

## 10. Добавить опросы (интерес/неинтерес)
**Заголовок:** Add polls for event interest (Interested/Not Interested)  
**Описание:** Внедрить кнопки для голосования по интересу к мероприятию с счетчиками.  
**Метки:** polls, engagement  
**Приоритет:** Low  

## 11. Внедрить уведомления (push и напоминания)
**Заголовок:** Implement notifications (FCM and local reminders)  
**Описание:** Добавить push-уведомления для обновлений и локальные напоминания за 2 часа до события.  
**Метки:** notifications, reminders  
**Приоритет:** Medium  

## 12. Добавить счетчик просмотров мероприятий
**Заголовок:** Add view counter for events  
**Описание:** Отслеживать и отображать количество просмотров каждого мероприятия для анализа популярности.  
**Метки:** analytics, views  
**Приоритет:** Low  

## 13. Реализовать управление клубами
**Заголовок:** Implement club management and directory  
**Описание:** Добавить справочник клубов, где администраторы управляют своими клубами и мероприятиями.  
**Метки:** clubs, management  
**Приоритет:** Medium  

## 14. Добавить экспорт данных в CSV
**Заголовок:** Add CSV export for attendance and other data  
**Описание:** Позволить экспортировать списки участников, посещаемости и другие данные в CSV для отчетов.  
**Метки:** export, csv, reports  
**Приоритет:** Low  

Эти задачи можно скопировать и создать как issues в GitHub-репозитории. Каждая задача включает заголовок, описание и метки для удобства.
